LAZY UPDATE OPTIMIZATION
=========================

Material descriptor sets updated at most once per frame:

    updatedMaterial[materialIndex] tracks which materials already updated

    Entity 1 (Material 0):
        - Check: updatedMaterial[0] = false
        - Update: flushDirty(Material 0)
        - Set: updatedMaterial[0] = true

    Entity 2 (Material 0):
        - Check: updatedMaterial[0] = true
        - Skip: Don't update again

    Entity 3 (Material 1):
        - Check: updatedMaterial[1] = false
        - Update: flushDirty(Material 1)
        - Set: updatedMaterial[1] = true

Benefits: Material data synced once regardless of entity count using that material
